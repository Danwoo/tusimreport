### 실제 운영 시 고려사항

#### 1. API Rate Limiting
- **Naver News API**: 일 25,000건 (무료)
  - 5개 쿼리 × 10개 = 50건/분석
  - 일 최대 500종목 분석 가능
- **Tavily Search API**: 월 1,000건 (무료)
  - 5개 쿼리 × 10개 = 50건/분석
  - 월 최대 20종목 분석 가능 ⚠️

#### 2. 중복 제거 효과
- **예상 중복률**: 10-30%
- **실제 측정 필요**: 실제 API 테스트 후 확인

#### 3. 응답 시간
- **Naver**: 5개 쿼리 × 0.5초 대기 = ~2.5초
- **Tavily**: 5개 쿼리 × 1초 대기 = ~5초
- **총 예상 시간**: ~7-10초 (API 호출 시간 포함)

## 🚨 Critical Issues (즉시 해결 필요)

### 1. **백테스팅 완전 부재** 🔴🔴🔴

**문제**:
- "BUY 신뢰도 78%"의 실제 승률을 알 수 없음
- DCF 적정가가 과거에 얼마나 정확했는지 증명 없음
- AI 패턴 인식의 목표가 달성률 불명

**영향**:
- 중급 투자자: "백테스팅 없으면 그냥 참고만 하고 끝"
- 전문 투자자: "전문가는 숫자 없이 믿지 않음"

**해결 방안** (우선순위 1):
```python
# Phase 0-A: 백테스팅 시스템 개발
1. 과거 1년 데이터로 BUY/SELL 추천 검증
   - 승률: BUY 추천 후 3개월 뒤 목표가 달성률
   - 예: "BUY 신뢰도 70% 이상 종목 50개 중 32개 수익 (승률 64%)"

2. DCF Fair Value 정확도 검증
   - 과거 적정가 vs 실제 주가 비교
   - 평균 오차율 계산

3. UI에 백테스팅 결과 표시
   - "이 AI의 지난 1년 BUY 추천 승률: 68%"
   - "평균 수익률: +8.5%"
```

**개발 시간**: 3-5일
**영향**: 신뢰도 급상승 (중급/전문가 전환율 +50%)

### 2. **정보 과부하 (초급 투자자 이탈)** 🔴🔴

**문제**:
- 10개 전문가 분석 결과를 다 읽을 시간 없음
- "저는 그냥 사야 하는지 말아야 하는지만 알고 싶은데..."
- 전문 용어(DCF, PER, PBR, WACC)가 어려움

**영향**:
- 초급 투자자: 실사용 의향 **30%** (너무 복잡해서 포기)

**해결 방안** (우선순위 2):
```python
# Phase 0-B: 초급자 모드 (Simple Mode)
1. UI 최상단에 모드 선택 추가
   - 🟢 초급자 모드 (3분 빠른 분석)
   - 🔵 전문가 모드 (10분 상세 분석)

2. 초급자 모드 = 3개 핵심 에이전트만
   - sentiment_expert (뉴스)
   - financial_expert (재무)
   - advanced_technical_expert (차트)

3. 결론을 맨 위에, 크게, 명확하게
   ┌─────────────────────────────────────┐
   │  🟢 BUY (신뢰도 78%)                │
   │  3개월 목표가: 85,000원 (+13.3%)    │
   │  손절가: 70,000원 (-6.7%)           │
   │                                     │
   │  💡 한 줄 요약:                     │
   │  현재 저평가 상태, 3개월 내 12%     │
   │  상승 예상, 매수 추천               │
   └─────────────────────────────────────┘

4. 용어 설명 툴팁
   - "DCF" 위에 마우스 → "현금흐름할인: 기업의 미래 가치를 현재 가치로 계산"
   - "PER 15.2배" → "업계 평균 18배 대비 저평가"
```

**개발 시간**: 2-3일
**영향**: 초급 투자자 전환율 +40% (30% → 70%)
